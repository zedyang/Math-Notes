    \subsection{Standard Gumbel
Distribution}\label{standard-gumbel-distribution}

\begin{enumerate}
\def\labelenumi{(\roman{enumi})}
\tightlist
\item
  We sample with the inverse CDF: \[
  F^{-1}(u) = -\log(-\log(u)),~~u\in (0,1)
  \]
\item
  As instructed, let \(Y_n:=M_n-a_n\), the CDF of
  \(Y_n=\max(X_1, ..., X_n)-a_n\) is \[
  \begin{split}
  F_{Y_n}(y) &= P(M_n\leq y+a_n) = P\left(\bigcap_{i=1}^n(X_i \leq y+a_n) \right) \\
  &= \prod_{i=1}^n P(X_i \leq y+a_n) \\
  &= \exp(-ne^{-(y+a_n)})
  \end{split}
  \] If \(Y_n\) is to be of standard Gumbel distributed, we need
  \(F_{Y_n}(y) = \exp(-ne^{-(y+a_n)})=\exp(-e^{-y})\) \(\Rightarrow\)
  \(a_n = \log n\).
\end{enumerate}

This implies that \(\{M_n\}\) grows in a rate of \(\log n\). Because $M_n - \log n$ is standard Gumbel distributed
